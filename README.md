## Overview

A Streamlit dashboard that provides **B2B market intelligence** for CTOs, investors, and recruiters. We don't just tell you who is hiring ‚Äî we tell you **how much it costs** and **how hard it is to hire**.

### The Core Insight

In IT and Engineering, the **skill is in the title**:
- ‚ùå Vague: "Sales Manager" (what do they sell?)
- ‚úÖ Specific: "Senior Python Developer", "Civil Engineer (Tunnelling)", "AWS Cloud Architect"

By applying **regex pattern matching** on job titles, we extract 11 tech stack categories ‚Äî bypassing the missing job description limitation.

### What's New in v4: The Talent Scarcity Index

Previous versions showed **Cost** (salary). v4 adds **Feasibility** (application counts) ‚Äî answering the most important hiring question: *"How hard and expensive is it to fill this role?"*

### The "Quadrant of Pain"

The killer feature ‚Äî a scatter plot mapping **Salary (Cost)** vs **Applicants (Ease of Hire)**:

- **Bottom Right (High Pay, Low Apps):** üî¥ "Danger Zone" ‚Äî Hardest to hire, most expensive. Action: Headhunt.
- **Top Left (Low Pay, High Apps):** üü¢ "Value Zone" ‚Äî Easy to find, affordable. Action: Post ads.
- **Bottom Left (Low Pay, Low Apps):** ‚ö†Ô∏è Niche roles with supply issues.
- **Top Right (High Pay, High Apps):** üí∞ Premium roles with healthy supply.

## Quick Start

### Prerequisites

```bash
pip install streamlit pandas plotly
```

### Step 1: Process Your Data

Place your raw CSV (1M+ rows) in this folder, rename to `tech_talent_radar.csv`, then run:

```bash
python data_processor_v4.py
```

This will:
1. Parse the JSON `categories` column
2. Filter for **IT & Engineering** rows only
3. Rename columns to standard names
4. Tag each job with a **Tech Stack** via regex
5. Extract **application counts** (`metadata_totalNumberJobApplication` ‚Üí `num_applications`)
6. Clean salary data
7. Output `tech_talent_radar_final_v4.zip`

### Step 2: Run the Dashboard

```bash
streamlit run app.py
```

The raw dataset (`tech_talent_radar.csv`, ~256MB, 1M+ rows) is too large for GitHub. To run locally:

3. Run `python data_processor_v4.py` to generate the ZIP
4. The processed `tech_talent_radar_final_v4.zip` is committed to the repo when available

### About `app_recommender.py`

This is a **separate B2C tool** (job seeker facing) that provides AI-powered job recommendations with skill gap analysis. It requires:
- `backend/recommendation_engine.py` (not included)
- `SGJobData.csv` dataset

To run: `streamlit run app_recommender.py`

It is intentionally **not merged** into the main dashboard ‚Äî see Architecture Decision below.

## Architecture Decision: Why Not Merge the Recommender?

The `app_recommender.py` is a **B2C Job Recommender** (for job seekers). The main dashboard is a **B2B Market Intelligence Tool** (for CTOs & investors). Merging them would dilute the value proposition.

Instead, we **extracted the key insight** from the recommender ‚Äî application count data ‚Äî and built the **Talent Scarcity Index**, which serves the B2B audience far better.

## Methodology

1. **Filter:** IT & Engineering categories (parsed from JSON)
2. **Extract:** Regex scan titles for 11 tech stack patterns
3. **Enrich:** Application counts as a proxy for talent supply
4. **Analyze:** Salary √ó Scarcity quadrant analysis
5. **Visualize:** Interactive Plotly charts in Streamlit
